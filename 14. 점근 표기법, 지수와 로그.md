## 빅오 표기법

- 가장 큰 항 이외의 부분은 큰 영향을 미치지 않기 때문에 단항식

- 빅오 표기법의 곱
  - O(n<sup>2</sup> X n<sup>3</sup>) = O(n<sup>5</sup>)

- 빅오 표기법의 합
  - O(MAX(n<sup>2</sup>  + n<sup>3</sup>)) = O(n<sup>3</sup>)

### 크기비교

O(1) < O(log n) < O(n) < O(nlog n) < O(n<sup>2</sup> < O(2<sup>n</sup>) < O(n!)
